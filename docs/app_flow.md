# APPLICATION FLOW

## 1. Setup Flow
1. Create account (admin)
2. Create Contribution Plan
3. Add Members
4. Assign Members to Plans

---

## 2. Daily Usage Flow

### Step 1: Receive Payment
- Admin sees bank alert
- Enters payment into system

### Step 2: Allocation
- System suggests allocation
- Admin confirms or adjusts

### Step 3: Update Records
- Payment marked as allocated
- Member payment status updated

---

## 3. Contribution Cycle Flow

1. Start cycle
2. Members begin paying
3. Admin records payments
4. System tracks who has paid
5. When all paid → payout triggered
6. Recipient marked as “received”
7. Cycle continues

---

## 4. Reconciliation Flow

1. Admin opens dashboard
2. Views unallocated funds
3. System suggests distribution
4. Admin resolves mismatches
5. All plans balanced
